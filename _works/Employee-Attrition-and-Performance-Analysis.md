**Employee Attrition and Performance Analysis**

A data-driven project analyzing employee turnover to identify driving factors and improve retention. It utilizes Powe BI for dynamic dashboards and actionable insights.

### Tech Stack
*   **Tools:** Power BI, Excel
*   **Techniques:** Data Visualization, Trend Analysis, Dashboarding

### Key Features
*   **Attrition Factors:** Identifies key variables contributing to employee turnover.
*   **Dynamic Dashboards:** Real-time visualization of workforce metrics.
*   **Strategic Insights:** monitoring trends to optimize productivity and retention.
*   **Interactive Reports:** Drill-down capabilities for detailed analysis.
