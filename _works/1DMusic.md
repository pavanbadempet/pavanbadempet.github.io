**1D Music: Material Design Music Player**

A sleek, intuitive Android music player built with strict adherence to Material Design guidelines. It offers a premium listening experience with dynamic playback controls and smart library management.

### Tech Stack
*   **Language:** Kotlin, Java
*   **Architecture:** MVVM
*   **Design:** Material Design Components

### Key Features
*   **Material Aesthetics:** Visually stunning UI with smooth transitions and animations.
*   **Smart Queue:** Customizable queue management for on-the-go playlists.
*   **Dynamic Controls:** Comprehensive gesture-based playback control.
*   **Personalized Library:** Smart organization by artist, album, and genre.
